# 轻碳生活小程序 - 登录版

这是一个用于记录个人碳排、查看统计趋势、参与低碳排行和管理个人成就的微信小程序。

## 已有功能

- 碳排记录：按交通、饮食、用电分类记录每日碳排放量。
- 数据统计：查看周、月、季度总排放、日均排放、趋势图和分类占比。
- 减排建议：根据碳排主要来源生成低碳建议。
- 排行榜：基于云数据库 `rankList` 展示低碳排行。
- 我的页面：微信登录、昵称头像、等级进度、低碳树、徽章、打卡日历、每日目标和本地数据导出。

## 登录版改进

- 使用 `wx.cloud.callFunction({ name: 'login' })` 静默获取 openid。
- 将用户授权入口放到“我的”页面，避免启动时强制弹出资料授权。
- 云函数使用 `cloud.DYNAMIC_CURRENT_ENV`，跟随部署环境运行。
- 排行榜更新在云函数内读取当前用户 OPENID，不再信任前端传入的 openid。
- 本地碳排记录保存后会同步总碳排到云端排行。

## 云环境配置

在 `config.js` 中填写微信云开发环境 ID：

```js
module.exports = {
  cloudEnvId: '你的云环境ID'
}
```

如果留空，小程序会使用微信开发者工具当前选中的默认云环境。

首次使用前，请在微信开发者工具中上传并部署：

- `cloudfunctions/login`
- `cloudfunctions/updateRank`

云数据库需要创建集合：

- `userData`
- `rankList`
